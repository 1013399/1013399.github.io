    【台中和平】武陵四秀另二秀(DAY-2 品田山前峰、品田山)

    品田山前峰，海拔3442公尺，無基石，約在品田步道0.35K處，該山不在百岳之列，品田前峰和品田山間隔著品田前峰V型斷崖，登臨品田山必須小心。

    品田山，海拔3524公尺，又稱波秦西崙山，為武陵四秀之首，擁有台灣特殊的摺曲地質構造，遠觀品田山山形，猶如「品」和「田」兩字的堆疊，原有森林三角點已遺失，山頂視野無敵，山勢為百岳十峻之一，名列台灣百岳No.22(舊序)。

    走走日期：2019/04/07

    # 走走路線

    # →05:10輕裝出發往品田山

    →途中回望池有山，天空已經開始漸漸泛紅

    →緩步行走在箭竹草坡步道上

    # →回望池有山、桃山、喀拉業山

    →進入樹林

    →眺望雪山及圈谷

    →品田山前峰，約在品田步道0.35K處，後方山頭即為品田山

    →小心翼翼下V型斷崖

    # →下抵斷崖底部，約在步道0.28K處

    →往上攀爬上品田山

    # →到達品田山山頂

    →品田山山頂有360度視野，賞景、拍照後原路回

    →回到V型斷崖

    →回到品田山前峰

    →回到箭竹草坡，池有山、桃山在正前方

    # →活水源叉路，距離新達山屋約50公尺

    若新達山屋儲水桶無水可用時，可由此下切取水

    # →回到新達山屋，休息、整裝

    →休息30分鐘後，重裝下山

    →回到池有名樹

    →回到池有山第二登山口，取右回

    →回到池有山第一登山口，續直行

    →回到三叉路口，直往桃山，取右池有步道下山

    # →回到三叉營地，休息、點心

    →續行下山

    →過了步道3K後，步道趨緩

    →於步道2.1K休息、午餐，續行過了1.9K後，連續下急陡坡

    →回到步道1.2K平坦空地，休息

    →到達步道1K里程樁，進入緩坡，過步道0.7K後，轉為之字型步道

    →到達池有山登山口，接上瀑布步道(約3.6K處)，取右回

    # →回到步道1.5K松濤亭

    →回到步道1.1K桃山叉路

    # →回到武陵吊橋

    →回到武陵山莊

    →回到停車場，梳洗換裝

    相關信息

    走走時間：約9小時(含休息、午餐)

    高低落差：(不含上下起伏)

    • 新達山屋 → 品田山前峰 約上升260公尺
    • 品田山前峰 → 品田山 約下降20公尺後上升100公尺
    • 新達山屋 → 武陵山莊 約下降1280公尺
